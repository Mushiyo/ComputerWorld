解題雜記：
　　  解題思路請見程式碼中註解的地方。然後有一點我一定要該一下，那就是while的運行條件如果寫 n != -1 的話就會莫名奇妙的RE。我因為這個原因吃了約十個RE，在接近◢▆▅▄▃ 崩╰(〒皿〒)╯潰 ▃▄▅▆◣的臨界點時突然想到改成 n > -1 ，其他行都完全沒動，然後送出去後就AC了0.0，你說奇怪不奇怪。

Java： 
import java.util.Scanner;

public class u10499 {
 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);
  long n = input.nextLong();

  while (n > -1) {
   // the surface area of a sphere is 4πr^2
   // each part of the divided sphere will increase the surface area by
   // πr^2
   // , which is the area of the two hemicycle on each divided part.
   // so the profit is ((n * πr^2) / 4πr^2) * 100%, n ≧ 2
   // after simplified the equation, we got n * 25 %, n ≧ 2
   // NOTE that when n = 1, the profit is 0%

   if (n > 1) {
    System.out.println(n * 25 + "%");
   } else {
    System.out.println("0%");
   }

   n = input.nextLong();
  }
 }

}
