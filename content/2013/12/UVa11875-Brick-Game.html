解題雜記：
　　這題要求的是數列中的中位數。由於題幹中有提到數列己排序好，且一定只有奇數個數，所以直接輸出最中間的那個數就好，不用再判斷偶數個數時的情況。

Solution Description:
    This problem is asking to find the median of a sequence of numbers. Since the sequence is sorted and the length of the sequence is always odd, simply output the middlest number then you can get AC.

Java：
import java.util.Scanner;

public class u11875 {

 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);

  while (input.hasNext()) {
   int T = input.nextInt();

   for (int caseNum = 1; caseNum <= T; ++caseNum) {
    int N = input.nextInt();

    int halfN = (N / 2);

    int captainAge = 0;
    for (int i = 0; i < N; ++i) {
     int age = input.nextInt();

     if (i == halfN) {
      captainAge = age;
     }
    }

    System.out.printf("Case %d: %d\n", caseNum, captainAge);
   }

  }
 }
}
