解題雜記：
　　將起點設定為(0, 0)，面向90度的方向（也就是面向y軸正向），然後不斷的依照輸入所給的數值操作：向前d單位，然後順時鐘轉t度。假設目前的所在位置為(x, y)，面向w度方向，則往前d單位後，新的位置會是(x + d * cos(w), y + d * sin(w))；順時鐘轉t度後，會變成面向w-t度方向。注意<cmath>中cos跟sin預設傳入的參數是弳度(radian)不是度度(degree)，所以在用cos跟sin之前要先把度度轉成弳度。另外就是題目要求的是輸出最後位置的整數部份，這邊直接把位置轉換成int輸出就好。如果用printf("%.0f", x)這樣的方式輸出的話，結果會變成輸出四捨五入到整數位的x，就WA QQ。

C++： 
/*Filename: aoj0016.cpp
 *Author: Mushiyo
 */

#include <stdio.h>
#include <cmath>

using namespace std;

int main() {
 double x = 0, y = 0;
 int direction = 90;
 int d = 0, t = 0;

 while (scanf("%d, %d", &d, &t) == 2) {
  if (d == 0 && t == 0) {
   break;
  }

  double dx = d * cos(direction * M_PI / 180.0);
  double dy = d * sin(direction * M_PI / 180.0);

  x += dx;
  y += dy;

  direction -= t;
  direction %= 360;
  if (direction < 0) {
   direction += 360;
  }
 }

 printf("%d\n", (int)x);
 printf("%d\n", (int)y);

 return 0;
}
