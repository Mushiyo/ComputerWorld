解題雜記：
　　超級經典題A + B Problem，AOJ這邊其實不用開到long long也可以過，不過為了保險起見，我還是用了。

C++： 
/*Filename: aoj1000.cpp
 *Author: Mushiyo
 */
 
#include <iostream>
 
using namespace std;
 
int main(){
    long long A = 0, B = 0;
 
    while(cin >> A >> B){
        cout << A + B << endl;
    }
 
    return 0;
}
