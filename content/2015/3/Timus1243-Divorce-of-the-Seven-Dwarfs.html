Solution Description:
Mod the input number 7 and output the remainder. Since the input can be 1050, this is a big number problem. Java users can simply use the BigInteger class.

Java：
/* Filename: Timus1243.java
 * Author: Mushiyo
 */

import java.util.Scanner;
import java.math.BigInteger;

public class Timus1243 {

 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);
  BigInteger seven = BigInteger.valueOf(7);

  while (input.hasNext()) {
   BigInteger N = input.nextBigInteger();
   
   System.out.println(N.mod(seven));
  }
 }

}
