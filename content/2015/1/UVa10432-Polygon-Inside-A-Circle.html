Solution Description:
Connecting the center of the circle with the vertices of polygon, we can disassemble the regular n-gon into n isosceles triangles with the length of the two legs equal to the radius of the circle, which is r. The area of such triangle is \(\frac{1}{2} \times r \times r \times sin(\frac{360^\circ}{n})\). Hence the area of the polygon is \(n \times \frac{1}{2} \times r \times r \times sin(\frac{360^\circ}{n})\).

Java：
/* Filename: UVa10432.java
 * Author: Mushiyo
 */

import java.util.Scanner;

public class UVa10432 {

 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);

  while (input.hasNext()) {
   double r = input.nextDouble();
   double n = input.nextDouble();
   double rad = Math.toRadians(360 / n);
   
   System.out.printf("%.3f\n", n * r * r * Math.sin(rad) * 0.5);
  }
 }

}
