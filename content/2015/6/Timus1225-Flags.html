Solution Description:
The problem is very similar to Fibonacci sequence.
Let R, W and B be strips of red, white and blue respectively. Let F(N) be the number of strips with length N.
When N = 1, obviously we only have R and W. So we have F(1) = 2.
When N = 2, we have RW and WR, i.e. F(2) = 2.
When N ≧ 3, we can get all legal strips with length N in the following manners:
For strips with length N - 1, if it is ends with R, we place W next to it; else, place R to it.
For strips with length N - 2, if it is ends with R, we place BW next to it; else, place BR to it.
(Note all legal strips will only ends with R or W)
Combining strips get from these two ways, we have all the legal strips with length N.
For example, when N = 3, we have RW + R = RWR, WR + W = WRW, R+BW = RBW and W+BR = WBR, so we have 4 ways, and when N = 4, we have RWR + W = RWRW, WRW + R = WRWR, RBW + R = RBWR, WBR + W = WBRW, RW + BR = RWBR and WR + BW = WRBW, total 6 ways.

Hence, after formulating our discovers, we have:
F(1) = 2, F(2) = 2 and F(N) = F(N - 1) + F(N - 2), N ≧ 3
Note F(45) will large than 231 - 1, so it is better to use long instead of int when finding the value of F(N), otherwise overflow will happens during the computation.

Java：
/* Filename: Timus1225.java
 * Author: Mushiyo
 */

import java.util.Scanner;

public class Timus1225 {

 public static void main(String[] args) {
  long[] sol = new long[45 + 1];
  sol[1] = 2;
  sol[2] = 2;
  
  for(int i = 3; i < sol.length; ++i){
   sol[i] = sol[i - 1] + sol[i - 2];
  }
  
  Scanner input = new Scanner(System.in);

  while (input.hasNext()) {
   int N = input.nextInt();
   
   System.out.println(sol[N]);
  }
 }

}
