Solution Description:
Enumerate the computer programmer's house number until we find ten pairs of numbers satisfied the property. Assume the houses is numbered from left to right and the programmer's house number is n. The sum of house numbers of her left neighbors is:
\[\frac{(1 + (n - 1))(n - 1)}{2}\]
Now, assum the last house number is \(n_{last}\), then the sum of house numbers of her right neighbors is:
\[\frac{(n_{last } + (n + 1))(n_{last } - (n + 1) + 1)}{2} \\
= \frac{n_{last }^2 - (n + 1)^2 + n_{last } + (n + 1)}{2}\]
And we wish to find a \(n_{last }\) which staisfy sum of left = sum of right:
\[\frac{(1 + (n - 1))(n - 1)}{2} = \frac{n_{last }^2 - (n + 1)^2 + n_{last } + (n + 1)}{2}\]
After rearranging the equation, we have:
\[n_{last }^2 + n_{last } - (2 \times \text{sum of left} + (n + 1)^2 - (n + 1)) = 0\]
If \(n_{last }\) exist, it will be an integer and its value is equal to:
\[\frac{-1 + \sqrt{1 - 4(- (2 \times \text{sum of left} + (n + 1)^2 - (n + 1)))}}{2}\]
Use the above equation to find the pairs.

Java：
/* Filename: UVa138.java
 * Author: Mushiyo

import java.util.Scanner;

public class UVa138 {

 public static void main(String[] args) {
  int countOutput = 0;
  int homeNum = 6;
  
  while(countOutput < 10){
   long sumLeft = sumLeft(homeNum);
   
   long rightNum = homeNum + 1;
   long c = - (2 * sumLeft) - rightNum * rightNum + rightNum;
   double possibleLast = (-1 + Math.sqrt(1 - 4 * c)) / 2;
   
   
   if(Math.ceil(possibleLast) == Math.floor(possibleLast)){
    System.out.printf("%10d%10d\n", homeNum, (int)possibleLast);
    ++countOutput;
   }
   
   ++homeNum;
  }
 }
 
 private static long sumLeft(int homeNum){
  return (1L + homeNum - 1) * (homeNum - 1) / 2;  
 }
}
