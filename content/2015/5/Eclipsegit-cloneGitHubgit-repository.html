title=使用Eclipse來git clone存放在GitHub上的git repository date=2015-05-22 23:26 full_date=2015-05-22T23:26:00.002+08:00 type=post allowComment=true tags=eclipse, 筆記, GitHub status=published ~~~~~~ 本篇旨在介紹如何用Eclipse來git clone存放在GitHub上的git repository
底下將以我的其中一個GitHub repository──uHuntJia來做為示範
uHuntJia的網址在此：https://github.com/Mushiyo/uHuntJia

1. 首先，打開你想clone的repository的頁面，在右欄找到"HTTPS clone URL"這行字，把下面的網址複製起來。


2. 打開Eclipse，選擇上方列的File，再選擇下拉式選單中的Import

3. 接著在跳出的視窗中選擇Git，再選Projects from Git，按下下方的Next

4. 選擇"Clone URI"，按下Next


5. 在URI欄位中填入步驟一中複製的網址。若網址沒錯，其他的Eclipse應該會自動幫你填好，因此這邊就不多描述（真是不盡責的作者？）。接著按下Next。

6. 接著選擇你想clone的branch。在本例中，由於只有master branch，所以可以不用選。接著按Next。

7. 在Directory欄位中，填入你希望clone下來的repository在本地端的儲存位置。填完按Next。

8. 選擇"Use the New Project wizard"，按下Finish。

9. 在跳出的視窗中，選擇你希望的Project類型。在本例中，我選的是general Project。選好後按下Next。

10. 接下來的步驟就如平常在開新Project那樣，故這邊就不再多詳加描述了。在本例中我只填了Project name後，就按下Finish。

11. 在步驟10按下Finish後，就可以在左側的Project Explorer看到clone下來的repo了，大功告成。

雖然本篇是以GitHub的repository為主題，但是只要把步驟1的地方改一改，凡是git repository都可以用這個方法clone下來喔。不妨試試看吧！
