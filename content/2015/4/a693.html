解題雜記：
　　假設我們有n個食物的飽足度\(f_1, f_2, \cdots, f_n\)。現在定義數列Sj，0≦j≦n，其中S0=0，而對於1≦j≦n，\(S_j = f_1 + f_2 + \cdots + f_j = \sum_{i=1}^{j}f_i \)。
不難看出當1≦l＜r≦n，
 \[ \begin{eqnarray}
S_r - S_{l-1} &=& (f_1 + f_2 + \cdots + f_r) - (f_1 + f_2 + \cdots + f_{l-1}) \\ &=& f_l + f_{l+1} + \cdots + f_r
\end{eqnarray}\]
因此，每當我們要計算第l個到r個食物的飽足度的總和時，我們只要計算Sr-Sl-1即可。

Java：
/* Filename: a693.java
 * Author: Mushiyo
 */
package BASIC;

import java.util.Scanner;

public class a693 {

 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);
  final int MAX_N = 100000 + 1;
  int[] food = new int[MAX_N];

  while (input.hasNext()) {
   int n = input.nextInt();
   int m = input.nextInt();
   
   for(int i = 1; i <= n; ++i){
    food[i] = food[i - 1] + input.nextInt();
   }
   
   while(m > 0){
    int l = input.nextInt();
    int r = input.nextInt();
    
    System.out.println(food[r] - food[l - 1]);
    
    --m;
   }
  }
 }

}
