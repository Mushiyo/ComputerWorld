Solution Description:
The warriors are organized in the form an equilateral triangle. Hence the problem is equals to: given n, find the max integer r which satisfied \( \frac {r(1 + r)}{2} \leq n\).
Since r must be a positive integer, after solving the equation, we have
\[ r = \lfloor \frac {-1 + \sqrt{1 + 8n}}{2} \rfloor \]
Then just use the equation dirctly :)

Java：
/* Filename: UVa11614.java
 * Author: Mushiyo
 */

import java.util.Scanner;

public class UVa11614 {

 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);

  while (input.hasNext()) {
   int testCase = input.nextInt();

   while (testCase > 0) {
    long n = input.nextLong();
    long row = (long) Math.floor((-1 + Math.sqrt(1 + 8 * n)) / 2);

    System.out.println(row);
    
    --testCase;
   }
  }
 }

}
