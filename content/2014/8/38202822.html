因數與倍數的基本性質
1. 若 a | b 且 b | a，則 a = ± b
2. 若 a | b 且 b | c，則 a | c
3. 若 a | b 且 a | c，則對所有整數m, n, 恆有a | (bm + cn)

證明：
1. ∵ a | b 且 b | a，可知存在整數 p, q，使得 b = pa, a = qb.
將a以qb置換，可得b = pqb. 因b≠0，故將b消去後得1=pq => q是1的因數 => q = ± 1 => a = ± b
2. ∵ a | b 且 b | c，可知存在整數 p, q，使得 b = pa, c = qb => c = qpa =>  a | c
3. ∵ a | b 且 a | c，可知存在整數 p, q，使得 b = pa, c = qa. 又 (bm + cn) = pam + qan，則明顯的 a | (bm + cn).

0的因數&倍數
1. 0不是任何整數的因數，只有在 a ≠ 0 的情況下，a | b 才有意義。
2. 0是任何非零數的倍數，也就是 a | 0 對所有非零整數 a 成立。

一些倍數判斷規則
1. 偶數（2的倍數）：個位數為偶數
2. 4的倍數：末二位為4的倍數
3. 8的倍數：末三位為8的倍數
4. 5的倍數：個位數為0或5
5. 3的倍數：各位數字和為3的倍數
6. 9的倍數：各位數字和為9的倍數

證明5&6：
假設n為k位數，且的各位數字由高到低依序為\( a_{k-1}a_{k-2} \cdots a_1a_0 \)，故
\[\begin{eqnarray}
n &=& 10^{k-1}a_{k-1} + 10^{k-2}a_{k-2} + \cdots + 10a_1 + a_0 \\
  &=& (\underbrace{9 \cdots 9}_{k-1個9} + 1)a_{k-1} + \cdots + (99 + 1)a_2 + (9 + 1)a_1 + a_0 \\
  &=& (\underbrace{9 \cdots 9}_{k-1個9}a_{k-1} + \cdots + 99a_2 + 9a_1) + (a_{k-1} + a_{k-2} + \cdots + a_1 + a_0) \\
  &=& 9(\underbrace{1 \cdots 1}_{k-1個1}a_{k-1} + \cdots + 11a_2 + 1a_1) + (a_{k-1} + a_{k-2} + \cdots + a_1 + a_0)
\end{eqnarray} \]不難看出若n的個位數字和為9的倍數，則n也是9的倍數；若n為9的倍數，則因為 3 | 9，n也是3的倍數。

\( \sqrt{n} \)與n，還有質數檢驗的範圍
1. 若正整數 n 可分解成兩正整 a, b 的乘積，則 \( a \leq \sqrt{n} \) 或 \( b \leq \sqrt{n} \).
2. n 為合成數，則 n 必有一質因數 p 滿足 \( 2 \leq p \leq \sqrt{n} \)
3. 若正整數 n > 1，且不大於\( \sqrt{n} \)的質數都不是 n 的因數，則 n 必為質數.

說明：
1. 因為若 \( a > \sqrt{n} \) 且 \( b > \sqrt{n} \) 則 ab > n

最大公因數與最小公倍數
1. a 與 b 的最大公因數(greatest common divisor)記為
\[ gcd(a, b) 或 (a, b)\]
多個數\( a_1, \cdots , a_{n - 1}, a_n\)的最大公因數則記為
\[ gcd(a_1, \cdots , a_{n - 1}, a_n) 或 (a_1, \cdots , a_{n - 1}, a_n)\]
2. a 與 b 的最小公倍數(least common multiple)則記為
\[ lcm(a, b) 或〔a, b〕\]
多個數\( a_1, \cdots , a_{n - 1}, a_n\)的最大公因數則記為
\[ a_1, \cdots , a_{n - 1}, a_n 或〔a_1, \cdots , a_{n - 1}, a_n〕\]

輾轉相除法原理
假設 a, b, q, r 皆為整數，且 a = bq + r，則 (a, b) = (b, r)

證明：
若 d 為 a 與 b 的公因數，則可知 d | a 且 d | b => d | a - bq => d | r ，也就是 d 也是 b 與 r 的公因數。若 d 為 b 與 r 的公因數，則可知 d | b 且 d | r => d | bq + r => d | a，d 也是 a 與 b 的公因數。得證。
