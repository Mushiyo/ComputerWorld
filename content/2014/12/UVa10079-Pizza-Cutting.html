Solution Description:
Assume there are already n - 1 (n > 1) straight line cuts on the pizza and the maximum number of pizza pieces made by these cuts are an-1. Now, consider the nth cut, it needs to cross all the previous cuts to make the maximum number of pizza pieces, so we have \(a_n = a_{n-1} + n\). And we can get the closed form as follows:
\[\begin{eqnarray}

a_n &=& n + a_{n-1} \\
    &=& n + (n - 1) + a_{n - 2} \\
    &=& n + (n - 1) + (n - 2) + \cdots + 1 + a_0
\end{eqnarray} \]
Since a0 means the maximum number of pizza pieces when there is no cut, we have a0 = 1.
Replace a0 into the previous equation, we have:
\[\begin{eqnarray}

a_n &=& n + (n - 1) + (n - 2) + \cdots + 1 + 1 \\
       &=& \frac{(n + 1) \times n}{2} + 1
\end{eqnarray} \]
Just use the formula to compute n directly.
For more detail discussion of the problem, you can read section 1.2 of Concrete Mathematics:A Foundation for Computer Science by Ronald Graham, Donald Knuth, and Oren Patashnik.

import java.util.Scanner;

public class UVa10079 {

 public static void main(String[] args) {
  Scanner input = new Scanner(System.in);

  while (input.hasNext()) {
   long N = input.nextLong();
   
   if(N < 0){    
    break;
   }else{
    System.out.println((N * (N + 1)) / 2 + 1);
   }
  }
 }
