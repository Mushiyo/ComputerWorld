解題雜記：
　　用DP解的，複雜度是O(n)（如果用暴力法一段一段找，複雜度是O(n^2)，會TLE）。這題其實就是一維最大連續元素和這個經典題型，網路上能找到的資源很多，故這邊就不對解法多加描述了。

Solution Description:
    Solve by O(n) DP method. If solved by O(n^2) brute-force (in Java), it will be TLE. The problem is actually a classical problem called maximum contiguous subsequence sum.

Java：
import java.util.Scanner;

public class UVa10684 {

 public static void main(String[] args) {
  final int MAX_N = 10000;
  int[] bet = new int[MAX_N];
  Scanner input = new Scanner(System.in);

  while (input.hasNext()) {
   int N = input.nextInt();
   if(N == 0){
    break;
   }   
   
   for(int i = 0; i < N; ++i){
    bet[i] = input.nextInt(); 
   }
   
   int minJackpot = bet[0];
   int maxJackpot = bet[0];
   int jackpot = bet[0];
   for(int i = 1; i < N; ++i){
    jackpot += bet[i];
    minJackpot = Math.min(jackpot, minJackpot);
    maxJackpot = Math.max(maxJackpot, jackpot - minJackpot);
   }
   
   if(maxJackpot <= 0){
    System.out.println("Losing streak.");
   }
   else{
    System.out.println("The maximum winning streak is " + maxJackpot + ".");
   }
  }
 }
}
